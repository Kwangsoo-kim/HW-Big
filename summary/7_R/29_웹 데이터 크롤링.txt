// 웹 데이터 수집

	1. 정적 웹 크롤링 (rvest 패키지 사용) 
		- 1. 크롤링 할 사이트의 url을 가져온다.
		- 2. read_html(url,encoding='')함수를 사용, 해당 url에 텍스트를 데이터화 한다.
		- 3. 해당 페이지에서 원하는 부분의 데이터를 가져온다. 일반적으로 html로 만들어진 
		페이지기 때문에 , html_nodes()함수를 이용하여, 원하는 부분의 nodes를 가져온다
		- 4. 가져온 nodes에서 필요한 부분인 내용물만 가져올땐 html_text() 함수를 사용한다.
		- 5. 원하는 text를 가지고 온 후, gsub() 함수나 str_replace_all() 함수를 사용하여 text 데이터를 정제한다.
		- 6. 결과적으로 나온 데이터들을 조합하거나 원하는 파일 형식으로 저장,분석한다.

	2. 여러 페이지 웹 크롤링
		- 위 사항에 for문을 적용시키면 수월하게 진행 할 수 있다.
