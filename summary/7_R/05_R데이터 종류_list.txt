// list
	
	1. List : 복합 구조형의 벡터에 해당하는 데이터 타입 
		- 리스트는 다른언어에서 보는 해싱 또는 딕셔너리에 해당
		- (키,값) 형태의 데이터를 담는 연관 배열
		- 리스트는 list(키=값,키=값,..) 형태로 데이터를 나열
		- [index]의 형태는 각 값이 아니라 '(키,값)'을 담고 있는 서브 리스트를 반환
			ex) data <- list(name='아무개',age=100)
		- 리스트 항목 개수 확이 함수는 NROW(),length()를 이용한다.
	2. 데이터의 조회 : 리스트변수명$키 
			ex) data$name , data[[index]]
		 
	3. 데이터의 삭제 : NULL을 대체시켜 삭제한다. 
		 	ex) data$name <- NULL
