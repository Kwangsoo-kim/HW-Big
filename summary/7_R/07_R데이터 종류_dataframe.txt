// Dataframe
	
	1. 데이터 프레임 : 2차원 구조이며, 복합형 데이터 타입. 처리할 데이터를 
		엑셀의 스프레드시트와 같이 표 형태로 정리한 모습을 한다.
		각 열에는 관측값의 이름이 저장되고, 각 행에는 매 관측단위마다 
		실제 얻어진 값이 저장된다.
		행렬과의 차이점은 열단위로 고유한 타입을 가질 수 있다는 점이 있다.  
	
	2. 데이터 프레임 생성 및 구조확인
		 - 생성식 : dataname <-data.frame(vector1,vector2,vector3,...)
		 - 구조확인 : str(dataname)
	
	3. 데이터 프레임 열 추가 및 삭제
		 - 열 추가 : dataname$newColumn <- c(...)
		 - 열 제거 : dataname$newColumn <- NULL

	4. 데이터 프레임 합치기
		- cbind(a,b) : 열단위로 합쳐준다. 두 객체는 행의 수가 같아야한다.
		- rbind(a,b) : 행단위로 합쳐준다 . 두 객체는 열의 수가 같아야한다.
	
	5. 데이터 프레임 조회
		- [ ]와 색인을 이용하여 원하는 위치의 열과 행 정보를 조회 할 수 있다.
		- [행인덱스,열인덱스] 가 기본 베이스
		- 음수를 넣으면 해당열 제외하고 출력한다.
		- subset : 데이터베스에서 데이터를 추출하듯 비슷한 역할을 한다.
			여러 조건을 넣어 원하는 데이터를 출력할 수 있게한다.
			ex)subset(dataname,dataname$column=' ')
		- head(),tail() : 맨 처음, 맨 마지막 6개행을 조회하게 해준다. 
			ex)head(dataname,n) - dataname내 n개행이 조회된다.
