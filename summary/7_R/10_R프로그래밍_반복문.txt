// 반복문

	1. for  : 벡터의 데이터를 모두 소비할 때 까지 실행하는 반복문 
		ex) for (val in sequence){
			statement	
		    }
	
	2. while : 조건식이 참일 동안 실행하는 반복문
		ex) while(조건식){
			statement
		    }
	
	3. repeat : 코드 블록을 여러 번 반복하는데 사용. 루프를 종료시키는 조건 문장이 없다
		따라서 반복문 내부에 종료 조건을 명시하고 break문을 사용해 반복을 종료해야한다.
