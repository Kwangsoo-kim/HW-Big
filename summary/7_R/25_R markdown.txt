// R markdown

	1. R markdown : Markdown 문법을 이용해 R 데이터 분석 고서를 생성하는 기능
		- 코드와 결과물을 설명글과 함께 보여줄 수 있다.
		- 동일한 분석 과정을 거쳤을 때 동일한 분석 결과가 반복되어 나온다.
		- HTML, 워드 , PDF등 다양한 포맷으로 저장 가능하다.
		- 별도의 문서 작성 소프트웨어 없이 R로 보고서 작성이 가능하다.

	2. Markdown 문법 
		-   ```{r}
			 ```사이에 코드를 입력하면 실행한 결과물이 출력된다.
			   별도의 이미지 필요없이 그래프도 출력 가능하다.
		     ```
		- 문자 앞뒤로 *특수 문자*을 넣으면 이텔릭체
		- 문자 앞뒤로 **특수 문자**을 넣으면 볼드체
		- 문자 앞뒤로 ~~특수 문자~~을 넣으면 취소선
		- 문자 앞뒤에 [특수 문자](http://www.google.com/search?q=special+character)를 
		넣으면 하이퍼링크
		- #,##,###,#### 으로 타이틀 크기 조정 가능
