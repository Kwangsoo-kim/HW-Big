// 데이터 테이블

	1. 데이터 테이블 : 데이터 프레임에서 상속. 데이터 프레임을 대신하여 사용.
			상당히 빠른 속도증가, 소스의 단순화가 가능.
		- fread()함수를 사용하여 데이터를 로드할때 에티어 테이블로 가져올 수 있다.
		- data.table()함수를 사용하여 데이터를 데이터 테이블로 변경가능.

	2. 데이터 부분집합만들기 
		- ex) dt[ i, j ,by, ...]
		- i : SQL에서 where문과 같다
		- j : SQL에서 select | update문과 같다
		- by : 그룹지을때 사용.
	3. 그외 기능
		- list() = .() 와 같다. 원하는 형태를 사용
		- 특정상황에서 문자열에도 앞에 - 를 붙이는 것이 가능하다.
		- keyby : 자동으로 결과가 그룹화 변수에 따라 오름차순으로 정렬
