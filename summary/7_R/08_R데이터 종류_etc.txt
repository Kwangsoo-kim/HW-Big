// 타입판별, 타입변환 , 문자열과 날짜

	1. 타입 판별
		- class( )  : 데이터가 어떤 타입인지 출력한다.
		- is.*( ) : 데이터가 * 타입인지 물어본다. TRUE 나 FALSE로 출력한다.

	2. 타입 변환
		- as.*( ) : *타입으로 데이터를 변환 시켜준다. 

	3. 문자열 
		- nchar() : 문자열의 길이를 나타내 준다. length( )는 벡터의 길이를 반환해준다.
		- paste() : 여러개의 문자열을 연결 시켜준다. 더블쿼테이션이 출력된다.
		- substr(string,start,end) : start에서 시작 end에서 끝나느 하위 문자열을 추출한다.
		- strsplit(stirng,delimiter) : delimiter를 기준으로 문자열을 분할한다. 
		- gsub(old,new,string : string의 모든 하위 문자열을 대체한다.
		- print() : 문자열에서 있는 특수문자까지 모두 출력
		- cat() : 문자열에 있는 특수문자를 출력하지 않고, 더블쿼테이션도 출력하지 않는다.
		- outer(문자열1,문자열2,paste,sep='') : 외적을 계산하기 위한 함수.
		- as.Date('2000-00-00','%Y-%m-%d') : 문자열을 날짜로 변환
