// aggregate ,table

	1. table : 데이터를 팩터로 묶을 수 있는 가능한 모든 쌍의 조합을 카운트 한 수의 표.
		막대그래프 또는 파이차트등을 통해 시각화 할 때 주로 사용.
		- ex) table(data)
	
	2. aggregate : 데이터를 하위 집합으로 분할하고 가각에 대한 요약 통계를 계산 한 다음
		결과를 반환.
		- ex) aggregate(x, by, FUN,...)
		- x : R 객체
		- by : 데이터 프레임 x의 변수와 길이가 같은 그룹화 할 요소의 목록.
