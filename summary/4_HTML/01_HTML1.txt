// HTML기본태그
	
	1. 글자 관련 태그  
		- h1 : 헤더를 의미하는 태그. 개행 포함. 뒤 숫자에 따라 크기가 바뀐다.
		- p : 본문에 사용되는 태그 단락 구분.
		- br : 행바꿈 태그.
		- a : 앵커태그 href 속성값을 지정해서 이동.
		- b : 볼드체
		- 그 외 다양한 태그들.
	
	2. 목록 태그 
		- ul : 순서가 없는 목록 태그
		- ol : 순서가 있는 목록 태그
		- li : 목록 요소
		- dl : 정의목록태그
		- dt : 정의 용어 태그
		- dd : 정의 설명 태그

	3. 표태그
		- table : 표형식 레이아웃 만드는 태그
		- caption : 표 제목
		- tr : 표 내부의 행 태그
		- th : 행 내부의 열. 제목 셀 태그
		- td : 행 내부의 열. 일반 셀 태그

	4. 이미지 태그
		- img : 태그 시작
		- src : 이미지의 경로 지정
		- alt : 이미지가 없을 때 나오는 글자 지정
		- width ,height : 이미지의 높이 지정 (css 부분에 설정해주는 경우가 더 많다)

	5.오디오,비디오 태그
		- audio : 오디오 태그 시작
		- vidio : 비디오 태그 시작
		- src : 파일의 경로 지정
		- preload : 파일을 재생하기 전에 모두 불러올지 지정
		- autoplay : 파일을 자동 재생할지 지정 (브라우저마다 상이함)
		- loop : 파일을 반복할지 지정
		- controls : 파일 재생 도구를 출력할지 지정
