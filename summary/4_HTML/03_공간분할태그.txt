// 공간 분할 태그

	1. div(블록형식의 태그) 	- 하나의 블록형식의 태그이므로 개행포함.
	2. span(라인형식의 태그) 	- inline 형식으로 공간을 분할.
	3. 시멘틱 태그 	- 특정 태그에 의미를 부여해서 웹 페이지를 만드는 것 .
			유지보수에 효과적이다.
			- 시멘틱 태그 구조
				header : 해더
				nav : 내비게이션
				aside : 사이드에 위치하는 공간
				section : 여러 중심 내용을 감싸는 공간
				article : 글자가 많이 들어가는 부분
				footer : 꼬리말
