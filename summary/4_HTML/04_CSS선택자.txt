// CSS 선택자

	1. Tag 선택자 : 특정 태그를 선택하여 CSS속성을 적용할 수 있다.
	2. 전체 선택자 : * 선택자를 사용하면 문서 전체를 선택하는 의미가 있다.
			일반적으로 새로운 HTML 실행시 초기화 작업에 사용된다.
	3. id,class	 : 태그의 id, class를 이용하여 CSS 속성 설정 가능하다. id-#,class-.
	4. Tag,id,class 혼합	: 세 태그 혼합하여 CSS속성 설정 가능하다. 
	5. 속성 선택자 : Tag의 속성을 선택하여 CSS속성을 적용할 수 있다.
		- 선택자[속성] : 지정한 속성을 가지고 있는 요소를 찾아 스타일을 적용
		- 선택자[속성=값] : 속성 값이 특정값과 같은 경우를 찾아 스타일을 적용
		- 선택자[속성^=값] : 속성 값이 지정한 문자로 시작하는 속성값에 대해서만 스타일을 적용
		- 선택자[속성$=값] : 속성 값이 지정한 문자로 끝나는 속성에 대해서만 스타일을 적용
	6. 후손 및 자손 선택자 : Tag의 속성을 선택하여 CSS의 속성을 설정할 수 있다.
	7. 동위 선택자 : 태그의 동등한 위치를 판단하여 CSS속성을 설정할 수 있다. 
		- + ; 하나만선택
		- ~ ; 다 선택
	 
	8. 반응 선택자 : hover상태에 따라 속성을 설정할 수 있다.
	9. 상태 선택자 : 상태에 따라 CSS 속성을 설정할 수 있다.
	10. 구조 선택자 : 구조에 따라 CSS 속성을 설정할 수 있다.
	13. 문자 선택자 : 특정문자 또는 문자열을 선택 CSS속성을 설정할 수 있다.
	14. 링크 선택자 : 링크 되어있는 문자를 선택하여 cSS 속성을 설정할 수 있다.
	15. 부정 선택자 : 나를 제외한 모든 태그에 CSS 속성을 적용할 수 있다.
