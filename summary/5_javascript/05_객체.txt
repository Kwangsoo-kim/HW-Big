// 객체

	1. 객체 개요 및 속성 과 메소드 : 일반적으로 배열 내부에 있는 값을 요소, 객체 내부에 있는 값은 속성이라한다.
		JavaScripit에서는 요소와 속성을 구분하지 않고 쓴다. 
		객체 속성 중 함수형 자료형인 속성을 메소드라고 부른다.
	
	2. 객체와 반복문 : 배열은 단순 for 반복문과 for-in 반복문으로 반복문 가능하나,
			객체는 단순 for 반복문으로는 불가능하고, for-in 또는 forEach()를 사용한다.

	3. 객체관련 키워드
		- in : for키워드와 별도로 사용하면 해당 키가 객체안에 있는지 확인할 수 있다. 
		- with : 복잡하게 사용해야 하는 코드를 짧게 줄여줄수 있는 키워드이다.
	
	4. 객체의 속성 추가와 제거
		 - 추가 : 객체명.속성 : 속성내용; 으로 추가가능
		 - 제거 : delete(객체명.속성); 으로 제거가능
