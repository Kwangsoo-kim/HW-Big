// 생성자함수

	1. 생성자 함수 : 객체지향 프로그램에서 객체를 생성할 때 호출되는 단위 메소드
	
	2. 프로토 타입 : 생성자 함수로 생선된 객체가 공통으로 가지는 공간으로 
		동일한 함수를 계속 생성하는 비효율적인 일처리를 줄임에 의의가 있다.

	3. 캡슐화 : 특정 속성이나 메소를 사용자가 즉각 바로 사용할 수 없게 막는 기술.
		Java와 달리 private같은 특성이 없기때문에, 따로 만들어줘야 한다.	

	4. 상속 : JavaScript에서 상속은 없다. 그러나 Java에서 처럼의 상속을 비슷하게나마 
		따라할 수 있다.
