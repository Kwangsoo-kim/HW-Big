// Pandas

	1. fillna : 결측치를 어떤 특정값으로 바꿀 수 있게하는 함수
		- DataFrame.fillna( )로 사용
		- 결측치를 바꾸고싶은 값을 넣어 변경할 수 있다
		- method 에 ffill을 입력하면 결측치가 아닌 이전값으로 대체
		- method 에 bfill을 입력하면 결측치가 아닌 다음값으로 대체
	 	- value값에 딕셔너리로 각 컬럼값에 대응하는 특정값을 입력하면 컬럼별로 결측치를 대체한다.

	2. replace : 변경되길 원하는 데이터를 변경하고자하는 데이터로 바꿀 수 있게 해주는 함수
		- rseplace(to_value,new_value)
			- to_value : 변경전 데이터 값
			- new_value : 변경후 데이터 값
		- 인자로 리스트, 딕셔너리 모두 사용 가능하다.
		- to_value에 정규표현식 사용도 가능하다. 그땐 regex=True를 추가 해주어야한다.

	3. where,mask : 조건이 만족하는 요소는 그대로 출력해주는 함수 (반) mask
		 - 조건이 만족하면 값을 그대로 출력해주며 만족하지 못한값은 결측치로 대체해서 출력한다.
		 - 따로 만족하지 않을때 대체되어질 값을 지정해 줄 수 있다.
		 - mask는 where와 반대되는 내용을 반환해주며, 그외적인 환경설정은 똑같다.

	4. dropna : 결측치 있는 데이터를 누락시켜주는 함수
		- 따로 인자를 두지않으면 결측치가 있는 행 제거해서 반환한다.
		- 인자로 how='all'을 입력하면 행의 모든 열의 데이터가 전부 결측치인 행만 제거해서 반환한다.
		- 인자로 axis=1 은 결측치가 있는 열들을 제거하여 데이터를 반환하며, 기본값은 0이다.
		- 인자에 thresh =n 을 입력하면 결측치가 n개 이상인 행만 제거하여 데이터를 반환한다.

	5. astype : pandas의 dtype 속성을 변경해주는 함수이다. 
		- 사용시 주의사항은 원본값에도 영향을 줄 속성변경이라면 함수내에 copy=True를 기입해야
		원본데이터엔 영향을 주지 않는다.
