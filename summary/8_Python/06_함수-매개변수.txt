// 함수 - 매개변수

	1. 기본값을 갖는 매개변수
		- 기본값을 갖는 매개변수는 선택적으로 사용가능.
		- 기본값을 갖는 매개변수는 매개변수중에서도 맨 뒤에 와야함.
		- 함수 호출시 입력안해도 기본값으로 적용되어 함수가 작동함.

	2. 기본 변수를 갖는 매개변수
		- 기본변수가 스칼라 변수일때, 기본값은 함수가 정의되는 지점에서 한번만 평가된다.
		- 기본변수가 리스트,셋,딕셔너리 또는 개체일때는 주소가 할당되어 변수에 전달되기 때문에
		기본변수 값이 변경된다.

	3. 순서인수와 키워드 인수
		- 순서인수 : 반드시 순서적으로 와야하는 인수
		- 순서인수와 키워드 인수가 같이 올 때는 키워드 인수가 반드시 뒤에 온다

	4. 튜플 매개변수를 이용한 가변인수
		- 매개변수 앞에 *을 붙여 정의한다.
		- 인수들이 퓨틀에 저장되어 전달된다.
		- 가변 인수 앞에 0 개 이상의 일반 인수가 올 수 있다.

	5. 딕셔너리 매개변수
		- **name 형식의 최종 형식 매개 변수가 있으면 형식 매개 변수에 해당하는 것을 
		제외하고 모든 키워드 인수가 들어있는 딕셔너리를 받음

	6. 함수정의시 매개변수의  순서 
		- 순서인자 -> 튜플인자 -> 키워드인자 -> 딕셔너리인자 
		- 위 순으로 정의해야한다.
	
	7. 인수의 언패킹
		- 튜플 언패킹 : 튜플인수명 앞에 *붙이면 언패킹된다.
		- 딕셔너리 언패킹 : 딕셔너리명 앞에 **붙이면 언패킹된다.
