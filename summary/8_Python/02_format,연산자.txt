// format, 연산자

	1. format : 출력할 문자, 숫자, 날짜 데이터에 형식을 지정 
		- '{[인덱스]:[공백대체문자][정렬방법][자릿수][타입]}'.format(변수명)
		- 특징 	: {} 만 쓸 경우 변수의 수가 {}개수보다 적으면 오류
		 	: {}안에 인덱스를 쓰면 오류가 나지 않는다.
			: 숫자의 경우 오른쪽 정렬이 기본값, 문자의 경우 왼쪽 정렬이 기본값이다.
			: 공백대체문자를 지정할때는 반드시 정렬방법기호, 자리수를 표현해야한다.
			: 공백대체문자는 자리수 앞에 0보다 우선순위가 높다
			: 자리수에서 소수점으로 표현할때, 전체 자리수 자리에는 소수점도 포함된다.
	
	2. 날짜 출력 : 일반적으로 time의 localtime이나 datetime의 datetime을 가져와서 사용한다.
		-  %Y : 연도4자리, %y : 연도2자리, %m : 월, %d : 일
		   %H : 24시간 , %I : 12시간, %p : PM또는AM, %M : 분, %S : 초 
		- strftime 함수, format함수로 출력가능

	3. 연산자 : 일반적으로 다른언어와 유사하다. 특이점만 나열
		- ** : 제곱
		- // : 나누기의 몫
		- / : 몫만 나오는게 아니고 소수점까지 나온다. 결과는 실수
		- 논리의 반전은 !를 사용하지않고 not을 쓴다
		- 비트 연산자는 2진수 8진수 16진수 사용할때 이용
