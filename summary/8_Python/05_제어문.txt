// 제어문

	1. 조건문 : if 
		- 다른 언어에 있는 if와 사용법이 유사하다.
		- 다른 언어와 다르게 괄호로 영역을 지정하는게 아니고 들여쓰기로 나타낸다.
		- else if 를 elif를 줄여쓰는것 외에 다른 것은 없다.

	2. 반복문 : for -in
		- for 변수 in 나열할자료:
		  반복할 문장
		  반복할 문장
		  ..
		  (else):
		   변수에 값이 할당되지 않은 경우 한번 실행할 문장
		- else 구문은 선택사항이다.
		- if와 마찬가지로 들여쓰기로 구분
		- 변수에는 여러 개가 올 수 있다.
		- 나열할 자료에 리스트,튜플,딕셔너리,셋등이 올 수도 있다.
		- 나열할 자료 위치에 range( )함수 사용가능하다.
		- range( from ,to ,by) :  from : 시작수(생략가능),to : 반복수 ,by: 증감시킬 단위

	3. 반복문 : while 
		- while 조건:
		  반복할 문장1
		  반복할 문장2
		  ....
		  else:
		  조건이 거짓일 경우 실행할 문장<br>
		- else 구문은 선택사항
		- 무한 반복문에 빠지지 않도록 조건식이 바뀔수 있는 변수값을 수정하는 코드필요
		- break, continue 사용가능하다.
