// 객체와 클래스

	1. 객체와 클래스 
		- 구조 : class 클래스명(상속받을 클래스명):
			   함수나 변수 선언
		- 파이썬에서 객체는 클래스를 참조하지만 변수데이터 조작시 다른영역에
		객체의 새로운 영역이 생기면서 참조되는 클래스에서 바뀐 변수와 연결이 끊어진다.
		독자적인 객체가 된다.
		
	2. 변수와 메소드
		- 객체를 할당해 원하는 변수 데이터를 저장 할 수 있다.
		- 파이썬은 객체 고유의 변수만들기도 가능하다.
	
	3. 인스턴스 메서드
		- 클래스에서 정의한 메서드의 인자에 self를 넣어준다. 
		- 이렇게 되면 해당 클래스 단독으로 메소드 발동은 불가능해지나,
		클래스를 참조하는 객체는 메서드를 사용할 수 있다.
		- 객체가 갖는 변수에 인스턴스 메소드가 접근하기 위해 self를 붙여 
		객체 변수에 접근할 수 있게한다. Java의 this와 유사한 기능을한다. 
		- self만 쓸필요는 없지만, 권장된다.
