// Pandas

	1. apply( ): 데이터 프레임이나 시리즈에 각 열 또는 각 행에 함수 적용가능하게함.
		- DataFrame.apply(func,axis,result_type)
			- func : 각 열 또는 행에 적용할 함수
			- axis : 함수가 적용될 축. 기본값은 0 열 별로 함수 적용. 1 이면 각 행 별로 함수가 적용
			- 함수가 리스트 또는 스칼라를 반환 할 경우 result_type 에 broadcast를 넣어주면 
				출력타입을 원본데이터 그대로 출력해준다
			- 함수 중 한줄로 끝나는 lambda함수를 사용할 수 있다.	
	
	2. applymap( ) : apply와 작동방식은 유사하나, 데이터 프레임에서만 사용 가능하고 
		        각 요소별로만 작동한다는 점이 다르다. 이외의 사용법은 apply와 유사하다.
	
	3. map( ) : apply와 작동방식은 유사한, 시리즈타입의  벡터에서만 사용 가능하고
		  각 요소별로만 작동한다. apply( )와 유사하며, 딕셔너리를 map에 적용하면 
		  딕셔너리의 키별로 시리즈 값이 적용된다는 특이점이 있다.
