// 파일 i/O 프로그래밍 _ json

	1. json 데이터 출력
		- 딕셔너리 list를 json 파일로 저장할 수 있다. json 모듈을 불러와야한다. 
		- 저장에는 json.dump()함수를 이용하며, 데이터에 한글이 있을때 ensure_ascii=False를 입력하면
		한글이 깨지지 않는다.
		- indent = '\t' 를 입력하면 입력데이터에 적당한 간격이 생긴다.
	
	2. 튜플 리스트를 json데이터로 출력하는 방법
		- 튜플의 내용물이 되는 객체에 딕셔너리 로 변환하는 함수를 임의로 설정하여
		default 값에 입력하여 데이터타입을 딕셔너리로 만들면 json파일로 출력가능하다. 

	3. json 데이터 입력
		- json 파일을 딕셔너리 리스트 변수로 불러올 수 있다.
		- object_hook를 사용하여 딕셔너리 말고 json파일의 데이터를
		파이썬의 객체로 가져올 수 도 있다. 딕셔너리타입을 객체타입으로 전환하는 함수를
		임의로 설정할 필요가 있다.
