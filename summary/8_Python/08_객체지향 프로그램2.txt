// 메소드,생성자와 소멸자, 상속,재정의

	1. 메소드
		- 인스턴스 메소드 : 인자에 self를 꼭 삽입. 객체를 통해서 호출할 메소드.
		- 클래스 메소드 : @classmethod를 통해서 선언한 메소드. 인자에 cls를 꼭 삽입
		- static 메소드 : @staticmethod를 통해서 선언한 메소드. 객체를 통해서 클래스를 통해서 호출 가능

	2. 생성자, 소멸자
		- 생성자 함수 : _ _ init _ _ (self[,매개변수1,...]) 객체가 생성될 때 자동 실행
          		생성시 필요한 코드를 포함.변수 초기화 기능.
		- 소멸자 함수 : _ _ del _ _ (self) 객체가 소멸될때 자동 실행.
          			소멸시 필요한 코드를 포함. 인스턴스 객체의 레퍼런스 카운트가 0이 될 때 실행.
	
	3. 상속
		- class  자식클래스명(부모클래스명): 
		- 위와 같은 형태로 상속을 받는다. 부모 클래스의 모든 속성들을 자식 클래스로 물려줄 수 있다.
		- 자식 클래스의 생성자는 부모 클래스의 생성자를 이용하여 인스턴스 변수 초기화 가능

	4. 재정의 
		- 부모 클래스에서 정의한 함수를 자식 클래스에서 다시 정의하는 것을 말한다.
		- 함수 이름과 파라미터가 부모클래스의 함수와 같아야 한다.
