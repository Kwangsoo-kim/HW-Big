// 데이터 구조_list2

	1. 요소 수정
		- 인덱싱으로 데이터 수정 : 특정 인덱스지정하여 요소 변경
		- 슬라이싱으로 데이터 수정 : 지정한 범위의 리스트 항목을 다른 항목으로 한번에 바꿈
		바꾸려는 항목이 더 많거나 더 적어도 항목들을 일괄적으로 변경
		- 슬라이싱에서 by가 포함될 경우 바꾸려는 값의 개수가 맞아야 한다.

	2. 삭제하기
		- pop() : 가장 마지막 요소가 반환되고 삭제.
		- pop(n) : n번째 인덱스 요소 반한되고 삭제.
		- remove() : remove(data) 해당 데이터를 삭제.
		- del : 뒤에오는 데이터를 삭제.
		- clear() : 데이터를 전부 삭제.
	
	3. 정렬하기 
		- sort() : 기본 정렬 오름차순. 실행 후 리스트 자체가 바뀐다. reverse=True 입력하면
			내림차순 정렬이 된다.
		- reverse() : 리스트를 역순으로 나열해준다. 리스트 자체가 바뀐다.
	
	4. 리스트 복제
		- copy( ) : 복제된 새로운 객체를 생성.
		  ** = : 주소를 복사해서 같은 객체를 참조, 원본객체의 데이터가 변경되면
			복사객체의 데이터도 변경됨.
