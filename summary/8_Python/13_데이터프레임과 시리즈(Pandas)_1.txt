// Pandas

	1. Pandas 
		- 1차원 구조의 Series 와 2차원 구조의 DataFrame 을 제공
		- 부분집합 조회, 열 추가 및 제거, 병합, 데이터 구조 변경 등 데이터 전처리를 위한 많은 기능 제공
 	
	2. DataFrame 만들기 : pd.DataFrame함수로 사용
		- 딕셔너리 사용 : 딕셔너리를 이용해 데이터 프레임을 만들면 키가 열 이름이 된다.		
		- 리스트 사용 : 딕셔너리의 값으로 지정하거나 , np.c로 열단위 결합후 열이름 지정해서 사용한다.

	3. csv 파일 불러오기 
		- pd.read_csv(filepath,sep=' ',skiprows=[ ],comment='#',....)
		- skiprows를 입력하여 csv가져오기전에 제외할 행들을 정할 수 있다. 
		- comment='#'기입하여 주석행 지정할 수 있다.

	4. DataFrame 조회 
		- 데이터프레임.열이름 , 데이터프레임['열이름'] : 열이름에 . 또는 공백이 있을땐 후자사용.
		- loc를 이용한 이름 조회 
			- DataFrame.loc[행이름,열이름] : 열이름은 생략가능, 슬라이싱, 리스트 사용가능
		- iloc를 이용한 인덱스로 조회
			- DataFrame.iloc[행인덱스,열인덱스] : 슬라이싱, 리스트 사용가능
		- 조건으로 조회하기 
			- DataFrame.loc[행조건] : 행 조건에 맞는 모든 열을 반환
			- DataFrame.loc[행조건, 열리스트] : 행 조건에 맞는 지정한 열을 반환
			- DataFrame.loc[중복조건] : 중복조건에 맞는 모든 행,열 반환. &연산자 사용
