// Linux 명령어
	
	1. 텍스트 보기
		- cat : 짧은 텍스트 보기엔 용이
		- more : 긴 텍스트 볼때 사용
		- less : 긴 텍스트 볼때 사용, more와 다르게 왔다갔다 가능함

	2. cp : 파일/디렉토리 복사

	3. mv : 파일이동 및 이름변경

	4. rm : 파일삭제 및 디렉토리(-r) 삭제

	5. ln : 파일링크만들기	

	6. touch : 빈 파일 생성, 수정 시간변경

	7. grep -n : 내용검색
	  
	8. whereis 파일명 , find : 파일검색
